Copyrigth @ PhoenixProgramming

## Deploying to Heroku

Push code to Heroku servers:

```shell
git push heroku master
```

Open Heroku app in browser:

```shell
heroku open
```

View/stream server logs:

```shell
heroku logs --tail
```
